# My Personal Blog

Welcome to my personal blog dedicated to network programming! This blog serves as a platform for sharing insights, tutorials, and articles primarily focused on Java and JavaScript programming. 

## Project Overview

This project is built using a static site generator, allowing for a fast and efficient way to publish content. The blog features a minimalist design, ensuring that the focus remains on the content itself.

## Structure

- **Home Page**: The homepage provides an introduction and overview of the blog's purpose.
- **Blog Page**: This section contains various posts related to Java and JavaScript programming.
- **Personal Profile**: An about page detailing my background and interests in network programming.

## Posts

The blog includes at least 9 posts covering various topics in Java and JavaScript, written in Vietnamese. Each post aims to provide valuable information and insights for readers interested in these programming languages.

## Getting Started

To run this blog locally, follow these steps:

1. Clone the repository:
   ```
   git clone [repository-url]
   ```

2. Navigate to the project directory:
   ```
   cd my-personal-blog
   ```

3. Install Hugo (if not already installed):
   ```
   brew install hugo
   ```

4. Start the Hugo server:
   ```
   hugo server
   ```

5. Open your browser and go to `http://localhost:1313` to view the blog.
